Critério de avaliação:

1. Cliente (2 pontos):
   - uso da biblioteca correta
   - troca de mensagens seguindo o padrão especificado
   - uso do relógio lógico
2. Bot (1.5 ponto)
   - uso da biblioteca correta
   - troca de mensagens seguindo o padrão especificado
3. Broker, proxy e referência (1 ponto)
   - broker e proxy funcionando (0.5 ponto)
   - servidor de referência conforme documentação (0.5 ponto)
4. Servidor (4 pontos):
   - uso da biblioteca correta
   - troca de mensagens seguindo o padrão especificado
   - uso do relógio lógico
   - sincronização de relógio
   - eleição de coordenador
   - sincronização de dados entre servidores
5. Documentação (0.5 ponto)
6. Apresentação (1 ponto): mostrar todos as partes funcionando

Avisos:

- projetos sem apresentação recebem nota zero de projeto
- caso seja encontrado partes de código iguais entre projetos, os grupos receberão nota zero
